# Viva Card Reader

## Requirements
 - Python 3.7+ 
 - SmartCard Reader

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements
```

## Usage

```bash
python main.py
```
